# 思维拆解 Skill · Debate Thinking Distiller

> Created by **赛博小熊猫Loki**

蒸馏辩论结构背后的认知操作序列。

底层是广义思维方式，载体是辩论逻辑。不蒸馏谁，蒸馏在压力下思考这件事本身。

## 两种模式

**拆解模式**：给你一份完整的思维拆解报告。六层操作体系逐层分析：切入角度 → 逻辑路径 → 立论 → 攻防 → 结辩 → 价值收束。

**陪练模式**：不给答案，只问你问题。三段式操作：问你 → 往深处探 → 给你一份复盘。帮你自己推出自己的立场。

## 理论依据

- 图尔明论证模型（Toulmin Model）六要素
- Paul-Elder 批判性思维框架
- 英国议会制（BP制）/ 奥瑞冈制 / CEDA 赛制结构
- 蒋昌建 / 黄执中 / 詹青云 / 马薇薇 / 庞颖 / 梁秋阳 / 胡渐彪 方法论提炼
- 第一性原理 / MECE / 金字塔原理 / 系统思维 / 假设检验 / 横向思维

## 参考书目

### 论证与逻辑理论

- Stephen Toulmin《The Uses of Argument》（论证的使用，1958）— 图尔明模型原著，六要素论证框架的起源
- Anthony Weston《A Rulebook for Arguments》（论证是一门学问）— 构建强论证的基本原则与常见谬误
- Richard Paul & Linda Elder《Critical Thinking: Tools for Taking Charge of Your Professional and Personal Life》（批判性思维）— Paul-Elder 批判性思维框架，9项智力标准
- Daniel Kahneman《Thinking, Fast and Slow》（思考，快与慢）— 系统一与系统二，认知偏误与决策判断
- 芭芭拉·明托《金字塔原理》— 结论先行、以上统下的结构化思维与表达方法

### 辩论体系与方法论

- 庞颖《思辨力35讲：像辩手一样思考》— 破（识别13种逻辑谬误）、立（9种分析框架）、场景重现三部分
- 黄执中《好好说话》系列 — 动机提问法、利他沟通、将是非题转化为选择题
- 梁秋阳《深度说服：超级辩手的四堂思辨表达课》— 观点思维，不带观点的思考都是无效思考
- 胡渐彪《青少年思辨课》— 30个可操作的思维方式，强调思辨力的日常应用
- 詹青云《像律师一样思考》— 法律思维在日常论证中的应用

### 思维模型与认知科学

- 查理·芒格《穷查理宝典》— 多元思维模型、跨学科思考框架
- 爱德华·德博诺《Lateral Thinking》（横向思维）— 打破线性逻辑，改变切入角度产生新视角
- 彼得·圣吉《第五项修炼》— 系统思维，反馈回路与杠杆点
- 纳西姆·塔勒布《黑天鹅》— 认知盲区、不确定性下的判断与决策
- Keith Stanovich《What Intelligence Tests Miss》（超越智商）— 理性思维与批判性思维的认知科学基础

### 赛事案例参考

- 1993年首届国际大专辩论赛决赛（狮城舌战）— 蒋昌建主论
- 国际华语辩论邀请赛（新国辩）历年决赛
- 《奇葩说》系列辩题
